nth-last-child

부모의 자식요소의 n번재가 해당 선택자와 일치하면 적용
일치하지 않을시 적용이 안될수도 있음

  • first-child : 부모의 자식 중에서 첫번째 요소
  • last-child : 부모의 자식 중에서 마지막 요소

nth-of-type

해당 선택자 요소중에서 n번째 선택

  • first-of-type : 해당 요소중에서 첫번째 선택
  • last-of-type : 해당 요소중에서 마지막 선택
  • nth-last-of-type : 뒤에서부터 n번째
